## 🎯 Description

Projet de reconnaissance de chiffres manuscrits utilisant un **Convolutional Neural Network (CNN)** implémenté avec TensorFlow/Keras. Le modèle atteint **99% d'accuracy** sur le dataset MNIST.

Ce projet a été réalisé dans le cadre d'un **stage en Machine Learning** et démontre les compétences en :
- Deep Learning & Computer Vision
- Prétraitement de données d'images
- Conception d'architectures CNN
- Évaluation et interprétation de modèles
